Colors

The color property in CSS is used to set the color of the text content of an element. It applies to all text inside the element, including headings, paragraphs, links, and other inline text content.

Syntax

selector {

color: value;

}


Where:

  • selector is the HTML element, class, or ID you're targeting.
  • value is the color you want to apply.

Color Values

You can define the color in several ways:

  1. Named Colors
    • Common color names, like red, blue, green, etc.
    • Example: color: red;

  2. Hexadecimal
    • A 6-digit code preceded by a #, where each pair represents red, green, and blue (RGB) values in hexadecimal.
    • Example: color: #ff5733;

  3. RGB
    • Defines color using the Red, Green, Blue color model, where each value is between 0 and 255.
    • Example: color: rgb(255, 0, 0); (This represents red.)

  4. RGBA
    • Similar to RGB, but with an additional A for alpha (opacity), where 1 is fully opaque and 0 is fully transparent.
    • Example: color: rgba(255, 0, 0, 0.5); (50% transparency red.)

  5. HSL
    • Defines color in terms of Hue (angle on the color wheel), Saturation (percentage), and Lightness (percentage).
    • Example: color: hsl(0, 100%, 50%); (Red).

  6. HSLA
    • Like HSL, but with an additional alpha value for opacity.
    • Example: color: hsla(0, 100%, 50%, 0.7); (Semi-transparent red.)

  7. CSS Variables
    • You can also use CSS variables for colors.
    • Example: --primary-color: #3498db; and then color: var(--primary-color);
