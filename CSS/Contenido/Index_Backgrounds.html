Backgrounds

The background property in CSS is a shorthand property used to style the background of an element. It combines multiple background-related properties into one declaration for concise and efficient styling.


Syntax

selector {

background: [color] [image] [position] [size] [repeat] [attachment] [origin] [clip];

}


Each value is optional and can be combined as needed.


Individual Background Properties:

  1. background-color
    • Specifies the background color.
    • Values: Named colors (red), Hex codes (#ff0000), RGB/A (rgb(255,0,0)), HSL/A (hsl(0,100%,50%))
    • Example: background-color: #f0f0f0;

  2. background-image
    • Sets a background image.
    • Values: URL (url('image.jpg')), none (default).
    • Example: background-image: url('background.jpg');

  3. background-position
    • Defines the position of the background image.
    • Values: Keywords (top, center, bottom, left, right), or percentages (50% 50%).
    • Example: background-position: center;

  4. background-size
    • Sets the size of the background image.
    • Values: Keywords (cover, contain), or dimensions (100px 200px, 50%).
    • Example: background-size: cover;

  5. background-repeat
    • Controls the repetition of the background image.
    • Values: repeat, repeat-x, repeat-y, no-repeat, space, round.
    • Example: background-repeat: no-repeat;

  6. background-attachment
    • Specifies whether the background image is fixed or scrolls with the page.
    • Values: scroll (default), fixed, local.
    • Example: background-attachment: fixed;

  7. background-origin
    • Determines the area within which the background is painted.
    • Values: border-box, padding-box, content-box.
    • Example: background-origin: padding-box;

  8. background-clip
    • Specifies how far the background extends.
    • Values: border-box, padding-box, content-box, text.
    • Example: background-clip: content-box;
