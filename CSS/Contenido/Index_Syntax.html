Syntax

CSS (Cascading Style Sheets) syntax is used to define the style of HTML elements on a webpage. It consists of the following components:


  1. Selectors

    2. These target the HTML elements you want to style. Common selectors include:

    • Element selector (p, div, etc.): Targets specific HTML tags.
    • Class selector (.className): Targets elements with a specified class attribute.
    • ID selector (#idName): Targets an element with a specified id attribute.
    • Attribute selector ([type="text"]): Targets elements based on attributes.

  2. Properties

    3. Inside a rule set, these define what aspect of the element is being styled. Examples include:

    • color: Sets text color.
    • font-size: Defines the size of the font.
    • margin: Specifies space around elements.
    • background-color: Sets the background color.

  3. Values

    4. These are the specific values that you assign to properties, such as:

    • red, #ff0000, rgb(255, 0, 0) (for color property).
    • 16px, 1em, 100% (for font-size or width properties).

Syntax Structure


A CSS rule is written in the following format:


selector {

property: value;

}


Example:

p {

color: blue;

font-size: 14px;

}


This CSS rule applies to all <p> elements, making their text blue and setting the font size to 14 pixels.


Additional Points


  • Comments: CSS comments are written as /* comment */, and are ignored by the browser.
  • Multiple properties: You can define multiple properties inside the curly braces, separated by semicolons.
  • Cascading: CSS allows styles to cascade, meaning more specific rules will override general ones. For example, an inline style will override an internal stylesheet.
