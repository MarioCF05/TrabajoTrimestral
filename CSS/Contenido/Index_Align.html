Align

In CSS, alignment properties control the positioning of elements and their content within a container. The align-related properties are used for text, inline elements, block elements, and flex/grid containers. They enable vertical, horizontal, and contextual alignment for both the element itself and its child elements.

Key Align Properties in CSS
  1. text-align

    2. Aligns inline content (like text) horizontally within a block-level container.

    • Values
      • left (default): Aligns content to the left.
      • right: Aligns content to the right.
      • center: Centers the content.
      • justify: Aligns content evenly across the width.
    • Example

      • p {

      text-align: center;

      }


  2. vertical-align

    3. Aligns inline or table-cell content vertically relative to the parent container.

    • Values
      • baseline (default): Aligns with the baseline of the parent element.
      • top, middle, bottom: Aligns to the respective positions.
      • Numeric values: Offset alignment (e.g., 10px).
    • Example:

      • img {

      vertical-align: middle;

      }


  3. align-items

    4. Aligns child elements vertically within a flex or grid container.

    • Values
      • stretch (default): Stretches items to fill the container.
      • flex-start: Aligns items at the start.
      • flex-end: Aligns items at the end.
      • center: Centers items.
      • baseline: Aligns to the baseline of the container.
    • Example

      • .container {

      display: flex;

      align-items: center;

      }


  4. align-self

    5. Aligns a single flex/grid child independently of others in the container.

    • Values: Same as align-items.
    • Example

      • .item {

      align-self: flex-end;

      }


  5. align-content

    6. Controls the alignment of multiple lines of content in a flex/grid container.

    • Values
      • stretch (default): Stretches lines to fill the container.
      • flex-start, flex-end: Aligns lines to the start or end.
      • center: Centers lines.
      • space-between, space-around, space-evenly: Distributes lines evenly.
    • Example

      • .container {

      display: flex;

      flex-wrap: wrap;

      align-content: space-between;

      }


  6. justify-content

    7. Aligns child elements horizontally in a flex or grid container.

    • Values
      • flex-start (default): Aligns items to the start.
      • flex-end: Aligns items to the end.
      • center: Centers items.
      • space-between, space-around, space-evenly: Distributes items evenly with varying spaces.
    • Example

      • .container {

      display: flex;

      justify-content: center;

      }


  7. place-items, place-content, and place-self

    8. These are shorthand properties for aligning items in grid containers:

    • place-items: Combines align-items and justify-items.
    • place-content: Combines align-content and justify-content.
    • place-self: Combines align-self and justify-self.
