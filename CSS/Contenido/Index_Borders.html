Borders

The border property in CSS is used to define the style, width, and color of the border around an element. You can use shorthand properties to combine all border aspects, or control each side (top, right, bottom, left) individually for precise styling.

Syntax

selector {

border: [width] [style] [color];

}


Each value is optional, but style is required to display a border.

Syntax

  1. border-width
    • Sets the thickness of the border.
    • Values: Keywords (thin, medium, thick), or specific sizes (1px, 0.5em).
    • Example: border-width: 2px;

  2. border-style
    • Defines the style of the border.
    • Values:
      • none: No border (default).
      • solid: A single solid line.
      • dashed: A dashed line.
      • dotted: A dotted line.
      • double: Two solid lines.
      • groove, ridge: 3D effects.
      • inset, outset: Other 3D effects.
    • Example: border-style: solid;

  3. border-color
    • Specifies the border's color.
    • Values: Named colors (red), Hex codes (#000), RGB/A, HSL/A, or transparent.
    • Example: border-color: blue;
Shorthand border

Combines border-width, border-style, and border-color into one property.


selector {

border: 3px dashed red;

}

Shorthand border

You can target specific sides of an element:

  • border-top
  • border-right
  • border-bottom
  • border-left

Example

selector {

border-top: 2px solid green;

border-left: 5px dotted blue;

}

Additional Border Properties

  1. border-radius
    • Creates rounded corners.
    • Values: Length (10px) or percentage (50% for circles).
    • Example: border-radius: 15px;

  2. border-image
    • Allows using an image as a border.
    • Values
      • border-image-source: URL of the image.
      • border-image-slice: Defines which part of the image is used.
      • border-image-repeat: How the image is repeated (stretch, repeat,round).
    • Example: border-image: url('border.png') 30 stretch;

  3. border-spacing
    • Applies to table elements to set space between borders.
    • Example: border-spacing: 10px;

  4. box-shadow
    • Although not technically a border property, it creates shadow effects around an element.
    • Example: box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.5);
