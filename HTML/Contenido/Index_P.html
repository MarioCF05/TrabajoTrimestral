Etiqueta <p>

The <p> tag in HTML is used to define paragraphs of text on a webpage. It is one of the most common and essential tags for structuring content.

Basic Definition
  • Represents a block of text grouped as a paragraph.
  • Used to separate ideas or blocks of textual content.

Semantic Structure
  • It is a block-level element, meaning it occupies the entire available width.
  • Enhances the document's semantics by clearly indicating where each paragraph begins and ends.

Default Appearance
  • Browsers add white space before and after each paragraph to improve readability.
  • This behavior can be adjusted with CSS.

Allowed Content
  • It can contain text and other inline elements, such as links (<a>), inline images (<img>), or formatted text (<strong>, <em>, etc.).
  • It should not contain block-level elements like other paragraphs or headings.

Accessibility
  • Improves accessibility by organizing content logically, making it easier for screen readers to process.
