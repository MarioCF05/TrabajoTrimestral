Etiqueta <a>

The <a> tag in HTML is used to define links or hyperlinks that allow users to navigate from one page to another or access resources on the web.

Main Features of the <a> Tag

  1. href Attribute (Hypertext Reference):

    • It is the most important attribute of the <a> tag. It defines the URL (web address) to which the link will lead.
    • Example: <a href="https://www.example.com">Visit Example</a>

  2. Link Text:

    • The text between the <a> </a> tags is what will be clickable on the page.
    • Example: <a href="https://www.example.com">Click here</a>

  3. target Attribute:

    • Defines how the link will open.
    • Common values:
      • "_self": Opens the link in the same window or tab (default).
      • "_blank": Opens the link in a new tab or window.
      • "_parent": Opens the link in the parent window or tab (if inside an iframe).
      • "_top": Opens the link in the entire browser window (removes frames).

    • Example:

      <a href="https://www.example.com" target="_blank">Open in a new tab</a>

  4. title Attribute:

    • Provides additional information about the link. This text appears as a tooltip when the user hovers the mouse over the link.
    • Example: <a href="https://www.example.com" title="Visit the example website">Visit</a>

  5. rel Attribute:

    • Specifies the relationship between the current document and the linked document.
    • Common values:
      • noopener: Improves security when opening links in new tabs.
      • noreferrer: Does not send referrer information to the destination site.
    • Example: <a href="https://www.example.com" target="_blank" rel="noopener">Open in a new tab</a>

  6. download Attribute:

    • Indicates that the link is for downloading a file rather than opening it in the browser.

    • Example:

      <a href="file.pdf" download>Download PDF file</a>
