Etiqueta <img>

The <img> tag in HTML is used to insert images into a webpage. It is one of the most important tags for adding visual content and enhancing the page's presentation.

Main Features of the <img> Tag:

  1. Basic Syntax:
    • The <img> tag is an empty element, meaning it does not have a closing tag. The src attribute is used to specify the source of the image.

    • Example:

      • <img src="image.jpg" alt="Description of the image">


  2. Common Attributes:

    • src (source):
      • Specifies the URL or path to the image file that will be displayed. It can be a relative or absolute path.
      • Example: <img src="path/to/image.jpg" alt="Description">
    • alt (alternative text):
      • Provides alternative text that describes the image. This is important for accessibility, especially for visually impaired users who use screen readers.
      • The alt attribute is also shown if the image cannot be loaded for any reason (e.g., if the image is broken or unavailable).
      • Example: <img src="logo.png" alt="Company Logo">
    • width and height:
      • Used to specify the dimensions of the image in pixels.
      • If not specified, the image will display at its original size.
      • Example: <img src="image.jpg" alt="Description" width="300" height="200">
    • title:
      • Provides additional text that appears when the user hovers the mouse over the image (a tooltip).
      • Example: <img src="image.jpg" alt="Description" title="Additional text">

  3. loading Attribute:

    • Specifies when the image should be loaded. There are two main options:
      • lazy: The image loads only when it comes near the visible area of the screen (lazy loading), improving performance.
      • eager: The image loads immediately when the page is loaded.
    • Example: <img src="image.jpg" alt="Description" loading="lazy">

  4. Accessibility:

    • The alt attribute is essential for improving accessibility, as it provides a textual description for users with visual impairments.
    • If the image is decorative and does not add relevant content, the alt attribute can be left empty: <img src="decorative.jpg" alt="">.

  5. Responsiveness:

    • Images should be responsive (adapt to different screen sizes). This can be achieved using CSS:

      • img {

      max-width: 100%;

      height: auto;

      }
