Lists

In HTML, lists are a way to organize content into grouped elements in a structured manner. There are three main types of lists.


Ordered Lists
  • Used when the order of elements is important, such as steps in a procedure or rankings.
  • Each item is automatically numbered by the browser.
  • Numbers can be customized with attributes.

Example:

<ol>

<li> Preheat the oven. </li>

<li> Mix the ingredients. </li>

<li> Bake for 30 minutes. </li>

</ol>


  1. Preheat the oven.
  2. Mix the ingredients..
  3. Bake for 30 minutes.

Attributes:

  • type: Changes the numbering style (for example, 1, a, A, i, I).
  • start: Specifies the starting number for the numbering.
Unordered Lists
  • Used to list items without a specific order, such as features, materials, or links.
  • Each item is marked with bullets by default.

Example:

<ul>

<li> Preheat the oven. </li>

<li> Mix the ingredients. </li>

<li> Bake for 30 minutes. </li>

</ul>


  • Preheat the oven.
  • Mix the ingredients.
  • Bake for 30 minutes.
Definition Lists
  • Ideal for presenting terms and their definitions, such as glossaries or specifications.

Structure:

  • <dl>: List container.
  • <dt>: Term to define.
  • <dd>: Definition of the term.

Example:

<dl>

<dt> Step 1 </dt>

<dd> Preheat the oven. </dd>

<dt> Step 2 </dt>

<dd> Mix the ingredients. </dd>

<dt> Step 3 </dt>

<dd> Bake for 30 minutes. </dd>

</dl>


Step 1
Preheat the oven.
Step 2
Mix the ingredients.
Step 3
Bake for 30 minutes.
Common Features:
  1. Nesting
    • Lists can contain other lists to create hierarchies.
  2. Accessibility
    • Add additional context using attributes like aria-label or aria-labelledby.
  3. Custom Styling
    • Use CSS to modify the layout.
