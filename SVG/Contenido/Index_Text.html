Text

In SVG (Scalable Vector Graphics), the <text> element is used to render text content in vector graphics. SVG text is fully scalable and can be styled, positioned, and manipulated for interactive or static purposes. It supports basic text rendering and advanced layout features.

Key Attributes of <text>

  1. x and y:
    • Define the starting position of the text.
    • Example: x="50" y="100"

  2. dx and dy:
    • Offset the position of the text relative to its current position.
    • Example: dx="10" dy="20"

  3. text-anchor:
    • Aligns the text relative to the x coordinate.
    • Values:
      • start (default): Aligns text to the starting point.
      • middle: Centers text at the x coordinate.
      • end: Aligns text to the ending point.
    • Example: text-anchor="middle"

  4. font-family:
    • Sets the font type.
    • Example: font-family="Arial, sans-serif"

  5. font-size:
    • Defines the text size.
    • Example: font-size="20px"

  6. fill:
    • Sets the text color.
    • Example: fill="blue"

  7. stroke and stroke-width:
    • Add an outline to the text.
    • Example: stroke="black" stroke-width="1"

  8. rotate:
    • Rotates individual characters around their baseline.
    • Example: rotate="45"

  9. lengthAdjust and textLength:
    • Control text scaling to fit a specific length.
    • textLength: Specifies the exact length.
    • lengthAdjust: Determines how text adjusts (spacing, spacingAndGlyphs).
    • Example: textLength="200" lengthAdjust="spacing"
Inline Elements for Advanced Text:
  1. <tspan>
    • Allows styling or positioning parts of the text independently.
    • Attributes: x, y, dx, dy, fill, font-size.
    • Example:

      • <text x="10" y="50">

      Normal text <tspan fill="red">styled text</tspan>

      </text>


  2. <tref>
    • References text content from another <text> element.
    • Example:

      • <text id="source">Reusable text</text>

      <text><tref xlink:href="#source" /></text>


  3. <textPath>
    • Positions text along a path.
    • Example:

      • <path id="curve" d="M10 80 C 40 10, 65 10, 95 80" fill="none" stroke="black" />

      <text>

      <textPath href="#curve">Text on a curve</textPath>

      </text>

Styling SVG Text:

SVG text can be styled using attributes or CSS:

  • Attributes:
    • fill="color", stroke="color", font-size="size", font-weight="bold|normal".
  • CSS

    • text {

    fill: blue;

    font-size: 16px;

    font-family: Arial, sans-serif;

    }
