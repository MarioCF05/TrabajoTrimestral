Marker

In SVG, the <marker> element is used to define reusable graphics that can be attached to the start, middle, or end of paths, lines, or shapes. It’s particularly useful for creating arrows, dots, or other embellishments on paths.

Syntax:

<marker id="markerID" refX="x" refY="y" markerWidth="width" markerHeight="height" orient="angle" [attributes]>

Graphic elements like shapes or paths

</marker>

Key Attributes of <marker>:
  1. id:
    • A unique identifier to reference the marker.
    • Example: id="arrowhead"

  2. refX and refY:
    • Define the reference point of the marker (typically where it connects to the path).
    • Values: Coordinates relative to the marker’s viewport.
    • Example: refX="0" refY="5"

  3. markerWidth and markerHeight:
    • Set the size of the marker's viewport.
    • Example: markerWidth="10" markerHeight="10"

  4. orient:
    • Determines the orientation of the marker.
    • Values:
      • auto: Rotates the marker to align with the path.
      • auto-start-reverse: Similar to auto, but reverses the marker at the start of the path.
      • A specific angle in degrees (e.g., 90 for a fixed angle).
    • Example: orient="auto"

  5. viewBox (optional):
    • Defines the coordinate system of the marker for scaling.
    • Example: viewBox="0 0 10 10"
Using <marker>with Paths:

Markers are applied to paths, lines, or other shapes using the marker-start, marker-mid, and marker-end attributes.

Example:


<path d="M10 10 L100 100" stroke="black" stroke-width="2" marker-end="url(#arrowhead)" />
