Image

In SVG (Scalable Vector Graphics), the <image> element is used to embed raster images (like JPEG, PNG, or GIF) within an SVG document. This allows SVGs to include non-vector elements while retaining their scalable properties and interactivity.

Syntax:

<image href="imageURL" x="x-coordinate" y="y-coordinate" width="width" height="height" [attributes] />

Key Attributes of <image>:
  1. href:
    • Specifies the URL or file path to the image.
    • Example: href="image.png"

  2. x and y:
    • Define the top-left corner's position of the image within the SVG canvas.
    • Default: 0
    • Example: x="10" y="20"

  3. width and height:
    • Set the dimensions of the image.
    • Default: The image's intrinsic dimensions are used if these attributes are not specified.
    • Example: width="200" height="150"

  4. preserveAspectRatio:
    • Controls how the image scales and fits within the specified width and height.
    • Values:
      • none: Distorts the image to fill the area.
      • xMinYMin, xMidYMid, xMaxYMax: Aligns the image to specific corners or the center while preserving aspect ratio.
      • meet: Scales to fit within the area while preserving aspect ratio.
      • slice: Scales to fill the area while preserving aspect ratio, possibly cropping the image.
    • Example: preserveAspectRatio="xMidYMid meet"

  5. opacity:
    • Sets the transparency level of the image.
    • Example: opacity="0.5"

  6. clip-path:
    • Clips the image using a defined shape or path.
    • Example:

      • <clipPath id="clip">

      <circle cx="50" cy="50" r="40" />

      </clipPath>

      <image href="image.jpg" clip-path="url(#clip)" />
