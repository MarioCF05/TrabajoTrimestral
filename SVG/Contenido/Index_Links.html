Links

In SVG (Scalable Vector Graphics), links can be created using the <a> element, which allows you to hyperlink parts of your SVG, such as shapes, text, or groups of elements. These links behave like regular HTML links and are clickable.

Syntax:

<a href="URL" [attributes]>

SVG content (shapes, text, groups, etc.)

</a>

Key Attributes of the <a> Element:
  1. href:
    • Specifies the URL or destination of the link.
    • Example: href="https://example.com"

  2. target:
    • Specifies how the link will open.
    • Values:
      • _self (default): Opens in the same tab.
      • _blank: Opens in a new tab.
      • _parent: Opens in the parent frame.
      • _top: Opens in the full body of the window.
    • Example: target="_blank"

  3. xlink:href (deprecated):
    • Previously used for linking in SVG but replaced by href in modern SVG.

  4. download:
    • Allows the linked file to be downloaded instead of opening.
    • Example: download="filename"
Accessibility and Styling:
  1. Tab Navigation:
    • Elements inside <a> tags are automatically focusable with the keyboard.
    • Ensure the linked content is descriptive for accessibility.

  2. Styling Links:
    • SVG links can be styled with :hover or :active pseudo-classes using CSS.
    • Example:

      • a:hover text {

      fill: orange;

      }


  3. ARIA Attributes:
    • Use ARIA attributes for better screen reader support.
    • Example:

      • <a href="https://example.com" aria-label="Go to Example">

      <circle cx="50" cy="50" r="40" fill="blue" />

      </a>
