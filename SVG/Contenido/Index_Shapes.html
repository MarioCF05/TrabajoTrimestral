Shapes

SVG (Scalable Vector Graphics) is an XML-based format for creating vector graphics directly in the browser. The <shape> elements in SVG are used to draw basic geometric shapes like rectangles, circles, ellipses, lines, and polygons. Each shape element has specific attributes for precise control over its appearance.

Key SVG Shape Elements and Their Attributes

  1. <rect>
    • Draws a rectangle or square.
    • Attributes:
      • x, y: Coordinates of the top-left corner.
      • width, height: Dimensions of the rectangle.
      • rx, ry: Rounded corners (optional).
    • Example:

      • <rect x="10" y="10" width="100" height="50" rx="10" ry="10" fill="blue" />


  2. <circle>
    • Draws a circle.
    • Attributes:
      • cx, cy: Coordinates of the center.
      • r: Radius of the circle.
    • Example:

      • <circle cx="50" cy="50" r="40" fill="red" />


  3. <ellipse>
    • Draws an ellipse.
    • Attributes:
      • cx, cy: Coordinates of the center.
      • rx, ry: Horizontal and vertical radii.
    • Example:

      • <ellipse cx="100" cy="50" rx="80" ry="40" fill="green" />


  4. <line>
    • Draws a straight line between two points.
    • Attributes:
      • x1, y1: Start point.
      • x2, y2: End point.
      • stroke: Defines the color of the line.
      • stroke-width: Sets the thickness of the line.
    • Example:

      • <line x1="10" y1="10" x2="100" y2="100" stroke="black" stroke-width="2" />


  5. <polygon>
    • Draws a shape with multiple sides (connected by straight lines).
    • Attributes:
      • points: A series of x,y coordinates for the vertices.
    • Example:

      • <polygon points="50,10 90,80 10,80" fill="yellow" />


  6. <polyline>
    • Similar to , but does not close the shape automatically.
    • Attributes:
      • points: A series of x,y coordinates.
    • Example:

      • <polyline points="10,10 50,50 90,10" stroke="blue" fill="none" />


  7. <path>
    • Draws custom shapes using a series of commands.
    • Attributes:
      • d: A string of commands (e.g., M for move, L for line, C for curve).
    • Example:

      • <path d="M10 10 H 90 V 90 H 10 Z" fill="none" stroke="purple" />

Styling Shapes

SVG shapes can be styled using attributes or CSS:

  • Fill:
    • fill="color": Fills the shape with a color.
    • Example: fill="orange".
  • Stroke:
    • stroke="color": Sets the outline color.
    • stroke-width: Sets the outline thickness.
    • Example: stroke="black" stroke-width="2".
  • Opacity:
    • fill-opacity or stroke-opacity: Controls transparency.
